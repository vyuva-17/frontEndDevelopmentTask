Task Summary

Summary of Task Implementation!

Technical Stack

Angular-9, Angular-material and sass.

Libraries & API

rxjs,HTTP, HTTP_Interceptor and CanActivate API with its dependents API are loaded using Dependency Injection.

Functionality

In this application I have Validated routing using logged user token(Can Activate), Handled request and response header and ajax error in one place for all services which invokes from rest services using rxjs libraries and HTTP Interceptor, Employee third party api is called in rest services and passed the data over component to render the employee data in child and displayed the child rendered data in parent component.
